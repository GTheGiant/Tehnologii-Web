Cel mai mare element al diagonalei principale

Problema: Se dă matricea A(m, m) cu elemente numere întregi. Să se afle cel mai mare element al diagonalei principale și să se afișeze la ecran linia care conține acest element.

Diagonala principală este formată din elementele A[i][i], unde i = 0, 1, ..., m-1.

Exemple rapide:

3x3 simplă | 4x4 cu valori negative | 5x5 cu valori mari

Explicația algoritmului

  1. Se citește matricea pătratică A de dimensiune m x m
  2. Se inițializează variabilele:
    • maxElement = valoarea primului element de pe diagonala principală (A[0][0])
    • maxRowIndex = 0 (indicele liniei care conține elementul maxim)
  3. Se parcurg elementele de pe diagonala principală (de la i=1 la m-1):
    • Dacă A[i][i] > maxElement, atunci:
      • maxElement = A[i][i]
      • maxRowIndex = i
  4. Se afișează maxElement și linia maxRowIndex a matricei

Complexitate: O(m) deoarece se parcurg doar elementele de pe diagonala principală.
