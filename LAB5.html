Deplasarea literelor spre stânga

Problema: Se dă un text. Să se deplaseze spre stânga cu o poziție fiecare literă, astfel încât litera de pe poziția n să fie plasată pe poziția n-1, iar prima literă să devină ultima.

Exemplu: Pentru textul "tractor" se va afișa textul "ractort".

Explicație: Toate literele se mută cu o poziție spre stânga. Litera de pe prima poziție se mută pe ultima poziție.

Exemple rapide:

Pași algoritm:

1. Citește textul
2. Verifică dacă textul are cel puțin 2 caractere
3. Extrage prima literă
4. Mută fiecare literă cu o poziție spre stânga
5. Pune prima literă la sfârșit
6. Afișează rezultatul

Rezultat

Vizualizarea deplasării:
↓
Rezultatul va apărea aici...

Explicația algoritmului

  1. Se citește textul de intrare
  2. Dacă textul are mai puțin de 2 caractere, se returnează textul original (nu are sens deplasarea)
  3. Se extrage prima literă a textului: firstChar = text.substring(0, 1)
  4. Se extrage restul textului fără prima literă: restOfText = text.substring(1)
  5. Se combină restul textului cu prima literă la sfârșit: result = restOfText + firstChar
  6. Se afișează rezultatul

Alternativ, se poate folosi un algoritm care parcurge fiecare caracter și îl mută pe poziția anterioară:

function deplasareStanga(text) {
    if (text.length < 2) return text;
    
    let result = "";
    
    // Mută fiecare caracter cu o poziție spre stânga
    for (let i = 1; i < text.length; i++) {
        result += text[i];
    }
    
    // Adaugă primul caracter la sfârșit
    result += text[0];
    
    return result;
}
