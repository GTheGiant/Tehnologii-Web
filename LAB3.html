Explicație: Se parcurge vectorul și se identifică cel mai mare element care este mai mic sau egal cu valoarea lui m.

Algoritmul:

  1. Se initializează o variabilă pentru rezultat cu o valoare inițială (ex: -Infinity)
  2. Se parcurge fiecare element din vector
  3. Dacă elementul este ≤ m și este mai mare decât valoarea curentă a rezultatului, se actualizează rezultatul
  4. La final, dacă rezultatul rămâne -Infinity, înseamnă că niciun element nu respectă condiția
